üé® Typography: Quicksand + Nunito

Quicksand (Headlines)

  • Main headline: 3.5rem, 700 weight
  • Video title: 1.8rem, 700 weight
  • Badge labels: 0.95rem, 600 weight
  • Feel: Playful, friendly, approachable
  • Rounded terminals: Kid-friendly without being childish
  • Perfect for attention-grabbing headlines

Nunito (Body Text)

  • Subheadline: 1.5rem, 400 weight
  • Badge reveal: 0.9rem, 500 weight
  • Section label: 0.9rem, 600 weight
  • Feel: Readable, warm, professional
  • Excellent readability: Perfect for longer text
  • Complements Quicksand's playfulness with stability

‚úì Why This Pairing Works

  • Both are rounded sans-serifs = visual harmony
  • Quicksand grabs attention, Nunito provides clarity
  • Perfect balance: playful + professional
  • Quicksand's personality, Nunito's readability
  • Both render Devanagari beautifully
  • Appeals to kids AND parents/teachers

üìù Implementation Notes

  • Both fonts free via Google Fonts
  • Fast loading (only 2 font families)
  • Weights needed: Quicksand 400,600,700 / Nunito 400,600,700,800
  • Headlines: font-family: 'Quicksand', sans-serif
  • Body: font-family: 'Nunito', sans-serif
  • Fallback: sans-serif (system fonts)

üéØ Usage Guidelines

Use Quicksand For:

  • H1, H2, H3 headlines
  • Section titles
  • Badge values/labels
  • Button text
  • Logo lockup (if needed)
  • Any text that needs personality

Use Nunito For:

  • Subheadlines
  • Body paragraphs
  • Lists and descriptions
  • Navigation text
  • Form labels and inputs
  • Any text that needs readability
