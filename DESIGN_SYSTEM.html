3. HTML Structure & Semantic Markup

Proper Heading Hierarchy (SEO Critical)

‚úÖ SEO Best Practice: Use only ONE H1 per page. Follow logical hierarchy: H1 ‚Üí H2 ‚Üí H3 ‚Üí H4. Never skip levels.

8. Animation Guidelines

Principles

  • Subtle, not distracting: Animations enhance, don't dominate
  • Quick and responsive: 0.26s for most interactions (feel snappy)
  • Purposeful: Every animation serves a function (feedback, delight, hierarchy)
  • Consistent timing: Use same duration across similar elements

Standard Transitions (Based on Section 2)

/* Button hover - color fill only */ transition: background 0.3s ease; /* Badge hover - grow + glow */ transition: transform 0.26s ease-in-out, box-shadow 0.26s ease-in-out; /* Badge icon slide */ transition: transform 0.26s ease-in-out, font-size 0.26s ease-in-out; /* Badge text fade */ transition: opacity 0.26s ease-in-out; /* Badge reveal fade-in */ transition: opacity 0.325s ease-in-out 0.065s; /* slight delay */ /* Card hover */ transition: transform 0.3s, box-shadow 0.3s;

Badge Hover Sequence (Section 2 Style)

/* 1. Badge grows and glows */ .badge:hover { transform: scale(1.05); /* or rotate + scale if angled */ box-shadow: 0 8px 30px rgba(color, 0.4); } /* 2. Icon slides up and shrinks */ .badge:hover .badge-icon { transform: translateY(-15px); font-size: 2rem; /* from 2.5rem */ } /* 3. Value and label text fade out */ .badge:hover .badge-value, .badge:hover .badge-label { opacity: 0; } /* 4. Reveal text fades in (with delay) */ .badge:hover .badge-reveal { opacity: 1; transition: opacity 0.325s ease-in-out 0.065s; }

What NOT to Do

  • ‚ùå No button slide/tilt on hover (just color change)
  • ‚ùå No excessive bouncing or spring animations
  • ‚ùå No auto-playing animations on page load (except subtle hero elements)
  • ‚ùå No animations longer than 0.5s (feels sluggish)
  • ‚ùå No animations on every element (creates chaos)
‚ö†Ô∏è Respect Motion Preferences: Some users have motion sensitivity. Always include:
@media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; } }
