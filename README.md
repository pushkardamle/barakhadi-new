# Barakhadi - Learn Marathi the Fun Way

A modern, responsive website for teaching Marathi language through interactive activities.

## Project Structure

```
Barakhadi-New/
├── index.html              # Main homepage
├── css/
│   └── styles.css          # Mobile-first stylesheet
├── assets/
│   └── images/
│       ├── barakhadi_favicon.png
│       ├── hero_image.png
│       ├── mainbarakhadilogo.png
│       └── product_name.png
└── README.md
```

## Features

- **Semantic HTML5**: Proper use of `<header>`, `<main>`, `<section>`, `<article>`, `<footer>` elements
- **Mobile-First CSS**: Responsive design optimized for all screen sizes
- **UTF-8 Encoding**: Full support for Marathi Devanagari script
- **Accessibility**: ARIA labels, semantic structure, keyboard navigation support
- **Performance**: Optimized images with lazy loading, clean CSS architecture

## Development

1. Open `index.html` in your browser
2. All styles are in `css/styles.css`
3. Images are in `assets/images/`

## UTF-8 and Marathi Support

The website is properly configured for Marathi language:
- HTML `lang="mr"` attribute
- UTF-8 charset declaration
- Devanagari text properly rendered in:
  - Level tabs (शिशु वर्ग, पहिली, दुसरी)
  - Activity cards (ड, ग)
  - Tagline image (चला शिकू या मराठी)
